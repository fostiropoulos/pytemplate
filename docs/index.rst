.. image:: _static/logo.png

Welcome to PyTemplate's documentation. Get started by reading our :doc:`usage`

The goal of this template project is to provide a good starting point of all the reasonable defaults when building your python project. Some of these include:

1. Documentation
2. Packaging
3. CI/CD actions
4. Static Analysis
5. Tests

When using this project you should modify the documentation to reflect your project's mission.
